# Architecture Overview

## System Components

### 1. Feature Extraction Layer
Processes raw transactions into 42-dimensional vectors.

### 2. Classification Engine
4-layer neural network optimized for ZKML.

### 3. ZKML Proof Generation
Generates zkSNARK proofs using EZKL.

### 4. MPC Integration
Middleware for custody platform integration.

### 5. Policy Enforcement
Risk-based approval workflows.
